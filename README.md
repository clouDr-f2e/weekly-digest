## 每周文摘阅读示例

```text
2020 # 年份
└── 07-04 # 7月第4周
    ├── README.md # 本周文摘
    └── 这是一篇原创文章.md # 原创文章
    └── 这是一篇翻译文章.md # 翻译文章（请标明出处）
    └── example # 文摘相关示例
    └── ...[other] # 其它自定义与文摘相关目录或文件
```
